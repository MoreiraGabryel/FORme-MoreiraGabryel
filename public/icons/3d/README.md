# 3D Tech Icons

Pasta reservada para os icones 3D do portfolio.

Objetivo:

- manter os icones 3D separados dos SVGs flat atualmente usados no site;
- organizar os assets por categoria;
- evitar mistura de arquivos antigos, testes e versoes finais.

## Estrutura

- `languages/` -> linguagens de programacao
- `frameworks/` -> frameworks e bibliotecas
- `tools/` -> ferramentas e plataformas

## Convencao de nomes

Use nomes curtos e consistentes:

- `python-3d.png`
- `java-3d.png`
- `spring-boot-3d.png`
- `react-3d.png`
- `typescript-3d.png`

## Regra de uso

Quando os icones 3D entrarem em producao:

- manter a pasta antiga `public/icons/` intacta ate a troca ser concluida;
- remover assets temporarios no mesmo bloco de trabalho;
- evitar nomes duplicados e versoes soltas como `final-final`.
