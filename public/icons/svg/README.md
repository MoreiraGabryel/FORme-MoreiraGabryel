# SVG Tech Icons

Pasta dedicada aos icones SVG que vao ser usados no portfolio com foco em:

- animacao
- leveza
- nitidez em qualquer escala
- compatibilidade com GSAP

## Estrutura

- `languages/` -> linguagens
- `frameworks/` -> frameworks e bibliotecas
- `tools/` -> ferramentas e plataformas

## Prioridade de uso

Se um icone precisar de:

- morph
- stroke animation
- fill animation
- glow controlado
- escalonamento sem perda

entao usar SVG.
